Threat Intelligence

Module Overview

This module focused on understanding the threat landscape, developing threat models, and using intelligence to anticipate and defend against cyber attacks. It emphasized structured analysis and tools such as MITRE ATT&CK, malware analysis, and threat hunting techniques.

Key Topics Covered

  • Threat modeling frameworks (STRIDE, DREAD)
  • MITRE ATT&CK framework
  • Open-source intelligence (OSINT)
  • Indicators of Compromise (IOCs)
  • Malware behavior analysis
  • Threat feeds and intelligence platforms

Skills Gained

  • Constructing and evaluating threat models
  • Using threat intelligence to support SOC operations
  • Identifying attacker TTPs
  • Correlating indicators with incident response processes

Assessment Highlight

The final assessment involved creating a detailed threat report analyzing a cyber incident using MITRE ATT&CK, including actor profiling, IOCs, and recommended mitigation strategies.
